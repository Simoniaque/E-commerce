Fonctions Utilisateur

checkLogin($con) : Vérifie si un utilisateur est connecté en se basant sur la session et retourne les données utilisateur.
getUserByID($con, $id) : Récupère les informations d'un utilisateur en fonction de son ID.
getUserInfo($con, $userId) : Récupère les informations détaillées d'un utilisateur à partir de l'ID utilisateur.
getUserAddresses($con, $userId) : Récupère les adresses associées à un utilisateur.
addUserAddress($con, $userId, $adresseComplete, $ville, $codePostal, $pays) : Ajoute une nouvelle adresse pour un utilisateur.
getUserPaymentMethods($con, $userId) : Récupère les méthodes de paiement d'un utilisateur.
addUserPaymentMethod($con, $userId, $type, $cardNumber, $cardName, $expirationDate, $cvv, $paypalEmail) : Ajoute une méthode de paiement pour un utilisateur.
editUserInfo($con, $id, $name, $phoneNumber, $address, $postalCode, $country, $city, $email) : Modifie les informations d'un utilisateur.
deleteUser($con, $id) : Supprime un utilisateur en fonction de son ID.

Fonctions Produits et Catégories

getProducts($con) : Récupère tous les produits.
getProductById($con, $productId) : Récupère un produit spécifique par son ID.
getCategoriesList($con) : Récupère toutes les catégories.
getCategoryById($con, $category_id) : Récupère une catégorie spécifique par son ID.
getProductsByCategory($con, $category_id) : Récupère les produits d'une catégorie spécifique.
getProductsByCategories($con, $idsCategories) : Récupère les produits appartenant à une liste de catégories.
getHighlightCategories($con) : Récupère les catégories mises en avant.
getHighlightProducts($con) : Récupère les produits mis en avant.
orderProductsByPrice($products) : Trie les produits par prix.
orderProductsByDate($products) : Trie les produits par date d'ajout.
filterProductsByMinPrice($products, $minPrice) : Filtre les produits par prix minimum.
filterProductsByMaxPrice($products, $maxPrice) : Filtre les produits par prix maximum.
getAllProducts($con) : Récupère tous les produits.

Fonctions Commandes

getOrdersByUserIdNewest($con, $user_id) : Récupère les commandes d'un utilisateur, triées par date décroissante.
getOrderDetails($con, $order_id) : Récupère les détails d'une commande spécifique.
getOrdersByUserIdAndNumber($con, $userId, $orderNumber) : Récupère une commande spécifique par ID et numéro de commande.
getOrderById($con, $orderId) : Récupère une commande spécifique par ID.
Fonctions Panier
getCart($con, $userId) : Récupère le panier d'un utilisateur.
getCartID($con, $userID) : Récupère l'ID du panier d'un utilisateur.
getCartDetails($con, $cartID) : Récupère les détails d'un panier spécifique.
getCartProducts($con, $cartID) : Récupère les produits dans un panier spécifique.
addToCart($con, $userId, $productId, $quantity) : Ajoute un produit au panier d'un utilisateur.
updateCartQuantity($con, $userID, $productID, $quantity) : Met à jour la quantité d'un produit dans le panier.
calculateTotalCartPrice($con, $userID) : Calcule le prix total du panier.

Fonctions Diverses

show_404() : Affiche une page 404 et interrompt le script.
generateURLVerifyAccount($con, $id) : Génère une URL pour la vérification du compte.
generateURLResetPassword($con, $id) : Génère une URL pour la réinitialisation du mot de passe.
checkPasswordResetToken($con, $userID, $token) : Vérifie la validité d'un token de réinitialisation de mot de passe.
resetPassword($con, $userID, $password) : Réinitialise le mot de passe d'un utilisateur.
userAlreadyExists($con, $email) : Vérifie si un utilisateur existe déjà avec un email donné.
addUser($con, $name, $email, $password) : Ajoute un nouvel utilisateur.
getUserByEmail($con, $email) : Récupère un utilisateur par email.
